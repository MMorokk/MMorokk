## 🏠 Home Lab

I run a homelab on a repurposed PC tunneled to a cheap VPS via WireGuard — born out of not wanting to touch my landlord's router.

- **Proxmox** for VM/LXC separation between stable services and experiments
- **Docker Compose** for everything that runs
- **Caddy** as reverse proxy + automatic SSL
- **CrowdSec + Fail2ban** keeping the VPS clean
